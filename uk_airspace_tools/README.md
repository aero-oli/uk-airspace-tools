# UK Airspace Tools QGIS Plugin

This folder is the installable QGIS plugin package.

For full installation, development, release, and safety notes, see the repository-level `README.md`.

## Main Capabilities

- Fetch and display UK NOTAM PIB XML.
- Fetch and display the NATS UK ICAO AIP Dataset for permanent airspace.
- Load local NOTAM XML and local AIP XML/KML/KMZ/ZIP datasets.
- Filter NOTAMs and permanent airspace by time, altitude, type, status, and keyword.
- Inspect NOTAMs from map clicks.
- Inspect permanent airspace from map clicks with a vertical cross-section diagram.

## QGIS Install

Copy or link this `uk_airspace_tools` folder into your active QGIS profile's `python/plugins` directory, then enable `UK Airspace Tools` in QGIS Plugin Manager.

This plugin is for visualisation only and is not a substitute for official briefing, NOTAM checking, regulatory review, or operational flight planning.
